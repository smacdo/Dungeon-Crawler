Building

Building SDL 1.3
-----------------
Ubuntu does not come with packages for SDL 1.3 (yet), so we must
manually build them. These instructions assume that you do not
already have SDL 1.2 installed (but it might work). Set up all your
build dependencies, and then checkout the SDL code somewhere

hg clone http://hg.libsdl.org/SDL
cd SDL
./autogen.sh
mkdir build
cd build
../configure 
make 
sudo make install 

hg clone http://hg.libsdl.org/SDL_image
cd SDL_image
./autogen.sh
mkdir build
../configure --disable-tif --disable-webp
make
sudo make install

hg clone http://hg.libsdl.org/SDL_ttf
cd SDL_ttf
./autogen.sh
mkdir build
../configure
make
sudo make install

hg clone http://hg.libsdl.org/SDL_net
cd SDL_net
./autogen.sh
mkdir build
../configure
make
sudo make install
